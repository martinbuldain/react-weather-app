## REQUERIMIENTOS

- Tener instalado node.js. Se puede descargar de [aquí](https://nodejs.org/es/).

## CÓMO INICIAR LA APLICACIÓN

- Abrir una terminal "cmd" dentro de la carpeta del proyecto "weather-react-frontend" y correr el siguiente comando para descargar las dependencias del archivo package.json:
```sh
npm install
```
- Luego correr el siguiente comando para iniciar la aplicación:
```sh
npm start
```
